# 隐私政策 / Privacy Policy

**Annotator**（以下简称"本扩展"）尊重并保护您的隐私。本政策说明本扩展如何处理您的数据。

## 我们收集的数据 / Data we collect
本扩展**不收集、不传输、不上传**任何个人数据或标注内容到任何远程服务器。

## 数据存储位置 / Where data is stored
所有标注记录、截图设置与界面偏好（主题、语言、截图最长边）仅保存在您本地浏览器的 `chrome.storage.local` 中，完全位于您的设备之上，仅本扩展可读写，不会被发送到任何外部服务。

## 权限说明 / Permissions
- `activeTab` / `scripting`：用于在您主动触发标注的页面注入内容脚本，实现元素高亮与标注。
- `storage`：用于在本机保存上述设置与标注。
- `clipboardWrite`：用于您点击"复制"时把提示词 / 截图写入系统剪贴板。
- `host_permissions` (`<all_urls>`)：用于在您选择标注的任意页面注入脚本；扩展不会在后台自动读取页面内容。

如有疑问，请通过项目的 GitHub 仓库或 Buy Me a Coffee 页面联系作者。
